# Slack Bot - Starsss
Welcome to starsss! To use, simply go into any chat on the HackClub organization, and use any one of these commands: /givemestar (which gives you a star), /starcat (which gives you a cat), /deletestar (which tells you no delete), /startest (tests to see if bot is working).
