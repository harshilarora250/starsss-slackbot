# Starsss

Welcome to the Starsss readme! You can use the bot anywhere in the HackClub workplace on slack.

## Commands

### `/givemestar`

Gives you a star! ⭐️

### `/starcat`

Gives you a cat! 🐈

### `/deletestar`

Why delete? No delete. Attempts to delete a star (you can't 🤣).

### `/starstest`

Test to see if the bot is working well.

## Usage

1. Invite Starsss to your Slack workspace.
2. Open any channel where the bot is available.
3. Run one of the mentioned slack commands.

## Tech Stack

* Node.js
* Slack API
