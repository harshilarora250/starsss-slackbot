# Starsss

Welcome and thank you for joining the Starsss readme. I'm pretty sure you can use the bot anywhere in the HackClub worksplace and in most channels.

## Commands that you can use!

### `/givemestar`

Gives you a star! ⭐️

### `/starcat`

Gives you a cat! 🐈

### `/deletestar`

Why delete? No delete. Attempts to delete a star (you can't lol 🤣).

### `/starstest`

Test to see if the bot is working well.

## Usage

1. Open any channel where the bot is available.
2. Run one of the mentioned slack commands.

## Tech Stack

* Node.js
* Slack API
